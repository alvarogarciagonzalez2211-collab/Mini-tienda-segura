# Mini Tienda Segura

## Características
- Registro de usuarios
- Inicio y cierre de sesión
- Publicación de productos
- Compra de productos
- Persistencia en archivo JSON local
- Validación de formularios
- Protección de rutas
- Hash de contraseñas con bcrypt
- Rate limiting
- Errores amigables

## Instalación
```bash
npm install
cp .env.example .env
npm start
